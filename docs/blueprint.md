# **App Name**: InterviewAce

## Core Features:

- User Authentication: Secure sign-up and login using Firebase Authentication (Email/Google).
- Interview Type Selection: Users can select the interview type (Technical, HR, Behavioral, Fresher).
- AI-Powered Mock Interviews: Conduct mock interviews using text and voice input, driven by AI question generation.
- Dynamic Question Generation: Leverage Gemini/Vertex AI to generate relevant and dynamic interview questions.
- Response Analysis: Analyze user responses for clarity, relevance, confidence, and communication quality, using a tool.
- Instant Feedback: Provide structured feedback on strengths, weaknesses, and suggestions for improvement.
- Performance Tracking: Store interview history, feedback, and performance scores in Firestore, displaying progress through intuitive graphs over time.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey trust and intelligence.
- Background color: Light gray (#F5F5F5) for a clean, distraction-free interface.
- Accent color: Teal (#009688) to highlight key information and CTAs.
- Body and headline font: 'Inter' for a modern, neutral, and readable feel.
- Use simple, professional icons from a consistent set.
- Employ a clean, intuitive layout with ample white space to reduce cognitive load.
- Use subtle animations for feedback and progress indicators.
